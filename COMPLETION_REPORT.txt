🚀 QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

1️⃣  Install Rust
    $ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2️⃣  Run the app
    $ cd mediaweb
    $ cargo tauri dev

3️⃣  Choose data directory on first launch

✨ That's it! Start uploading media & creating pages.


⌨️ KEYBOARD SHORTCUTS
═══════════════════════════════════════════════════════════════════════════════

Navigation:
  M           → Media section
  P           → Pages section
  S           → Settings section
  /           → Focus search bar
  Esc         → Close modal

Media:
  Ctrl+U      → Upload media
  Enter       → Add tag (in editor)

Page Builder:
  Ctrl+N      → New page
  Ctrl+S      → Save
  Ctrl+Shift+A → Add section

See SHORTCUTS.md for more.
