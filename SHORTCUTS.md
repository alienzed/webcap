# MediaWeb Keyboard Shortcuts

## Navigation

| Shortcut | Action |
|----------|--------|
| `M` | Go to Media section |
| `P` | Go to Pages section |
| `S` | Go to Settings section |
| `/` | Focus search bar |
| `Esc` | Close modal / Cancel edit |

## Media Management

| Shortcut | Action |
|----------|--------|
| `Ctrl+U` or `Cmd+U` | Upload media |
| `Enter` | Add tag (in media editor) |
| `Ctrl+S` | Save media metadata |

## Page Builder

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New page |
| `Ctrl+S` | Save page |
| `Ctrl+Shift+A` | Add section |
| `Delete` | Delete selected block |
| `↑` / `↓` | Move block up/down |

## Tips

- Click any tag to filter media
- Tags are auto-indexed as you edit
- Pages auto-save to local files
- All data is viewable as JSON in your editor
